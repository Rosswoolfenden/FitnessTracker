How To Start!
To set up, first download the app "OwnTracks" from IOS App Store or Android Google Play Then you must enable the location tracking services for app. Upon reaching the main page in the image, to set up the streaming service, you must go to the settings menu, indicated by "i" in the top left corner. After that, tap the settings button and change the parameters to match the ones in the image.
Now go back to the main page and change the data capturing mode to "move mode", indicated by 2 arrows. Once done, you must open the "Subsriber" file on your computer from where you have downloaded it, using python 3

That concludes the set up. For the calories burned graph to be more accurate, only have the OwnTracks app open while doing a run and close it after finishing.
