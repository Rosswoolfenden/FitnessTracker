How To Start!
To use the location tracker with an iPhone you will first need to download OwnTracks. Once you have downloaded the app and opened it you will be presented with this pop up, when you see this you MUST allow the app to use your location. It is best to select the "Only While Using the App" option. After you have clicked an option from the pop up you will be presented with this message from Own Tracks. It explains how the app will work and once you have read this you can then press continue. Once you have pressed continue a map will come up showing your location, however the data of your location isnt currently being recorded. To change the settings so your locaton can be tracked you need to press the I in the top left corner. You will then be taken to this page there you will then press settings which is located under the status section.
Once this page has loaded, you will then need to change the mode, it will originally be set to public and youll need to change it to private. After the mode has been changed the page will then look like this where other settings will need to be changed. First you will need to change the host to iot.eclipse.org. Secondly change the Port to 1883, and thirdly turn the TLS and Authentication OFF. After you have made all of these changes the page should look like this and the data of your location will now be streamed to the website.
